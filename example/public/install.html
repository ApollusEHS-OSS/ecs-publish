Installation¶

Prerequisites¶

  • An existing node project, with Dockerfile, using git
  • For running your tasks, an existing AWS ECS Cluster
  • For running web services, an existing Application Load Balancer
  • For creating/updating DNS, an existing Route 53 Hosted Zone
  • For performing deployments, AWS credentials, i.e.
    • EnvironmentCredentials, or
    • SharedIniFileCredentials, or
    • EC2MetadataCredentials

Install¶

npm install @cimpresscloud/ecs-publish --save-dev
